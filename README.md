# Sales Data Cleaning and Preparation using Power BI

## Overview
This project focuses on data cleaning and preprocessing of a sales dataset using Power BI. The main goal was to transform raw, inconsistent data into a structured and analysis-ready format.

## Problem Statement
The dataset contained missing values, inconsistencies, and potential data quality issues, making it unsuitable for direct analysis. The task was to clean and prepare the data for further analytical use.

## Objectives
- Identify and handle missing values  
- Clean and standardize dataset  
- Ensure consistency and correctness of data  
- Prepare dataset for future analysis and visualization  

## Tech Stack
- Power BI (Power Query)
- CSV dataset

## Data Description
The dataset includes sales-related information such as:
- Product data  
- Sales values  
- Possibly incomplete or inconsistent entries  

The data initially contained missing values and required preprocessing.

## Methodology

### 1. Data Loading
- Imported CSV dataset into Power BI  
- Inspected structure and column types  

### 2. Data Cleaning
- Identified missing values across columns  
- Handled null values (removal or transformation)  
- Removed or corrected inconsistent entries  

### 3. Data Transformation
- Standardized column formats  
- Adjusted data types (numeric, categorical, date if applicable)  
- Ensured consistency across dataset  

### 4. Data Validation
- Verified cleaned dataset  
- Ensured no critical data loss  
- Prepared data for downstream analysis  

## Results
- Cleaned and structured dataset ready for analysis  
- Improved data quality and consistency  
- Eliminated missing and invalid entries  

## Key Takeaways
- Data cleaning is a critical step before any analysis  
- Even simple datasets require preprocessing  
- Power BI Power Query is effective for quick data preparation  

## Possible Improvements
- Automate cleaning pipeline  
- Add data validation rules  
- Connect to live data sources instead of static CSV  
- Extend project with dashboard and analytics

# Очистка и подготовка данных продаж в Power BI

## Обзор проекта
Проект посвящен очистке и предобработке данных продаж с использованием Power BI. Основная цель — преобразовать "сырые" данные в структурированный и пригодный для анализа вид.

## Постановка задачи
Исходный датасет содержал пропущенные значения и несоответствия, что делало невозможным корректный анализ. Требовалось очистить и подготовить данные.

## Цели проекта
- Найти и обработать пропущенные значения  
- Очистить и стандартизировать данные  
- Обеспечить корректность и согласованность  
- Подготовить данные для дальнейшего анализа  

## Используемые технологии
- Power BI (Power Query)  
- CSV файл  

## Описание данных
Датасет содержит информацию о продажах:
- Данные о продуктах  
- Значения продаж  
- Пропуски и некорректные значения  

## Методология

### 1. Загрузка данных
- Импорт CSV файла в Power BI  
- Анализ структуры данных  

### 2. Очистка данных
- Обнаружение пропущенных значений  
- Обработка null-значений (удаление или замена)  
- Исправление некорректных записей  

### 3. Трансформация данных
- Приведение форматов колонок  
- Изменение типов данных  
- Стандартизация значений  

### 4. Проверка данных
- Контроль качества после очистки  
- Проверка целостности данных  
- Подготовка к анализу  

## Результаты
- Подготовленный и очищенный датасет  
- Улучшено качество данных  
- Устранены пропуски и ошибки  

## Ключевые выводы
- Очистка данных — обязательный этап аналитики  
- Даже простые данные требуют подготовки  
- Power BI удобен для быстрой предобработки  

## Возможные улучшения
- Автоматизация процесса очистки  
- Добавление правил валидации  
- Подключение к реальным источникам данных  
- Расширение проекта (дашборд, аналитика)
